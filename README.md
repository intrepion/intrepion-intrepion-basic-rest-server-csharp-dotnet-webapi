# intrepion-intrepion-basic-rest-server-csharp-dotnet-webapi

### Build

```bash
dotnet build
```

```bash
dotnet test
```

```bash
CLIENT_URL="<CLIENT_URL>" dotnet run --project IntrepionApp.WebApi
```
